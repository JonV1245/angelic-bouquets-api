# Angelic Bouquets API

REST API built with FastAPI (Python) for Angelic Bouquets. Includes a health check and customer CRUD endpoints with validation via Pydantic and interactive Swagger docs.

## Features
- Health check endpoint
- Customers CRUD (currently in-memory; DB-ready structure)
- Swagger UI docs auto-generated by FastAPI

## API Docs (Swagger)
- Swagger UI: http://127.0.0.1:8000/docs
- OpenAPI JSON: http://127.0.0.1:8000/openapi.json
